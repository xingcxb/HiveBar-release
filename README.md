# `HiveBar` (`Release`)

> `HiveBar` 是一款轻量级 `macOS` 菜单栏工具：安静常驻、随用随开，帮助你更快完成日常开发 / 运维 / 系统操作。

## ✨ 特性
- 菜单栏常驻：不打扰、随时可用
- 轻量快速：启动快、占用低
- 本地优先：尽量在本机完成处理（更安全、更可控）

## ✅ 已支持功能
- **历史 `Shell` 命令**：读取并检索历史命令，便于复用
- **`Hosts` 管理**：查看与修改 `hosts` 文件（需管理员权限确认）
- **本机 `IP`**：快速查询本机网络/`IP` 信息
- **局域网文件分享**：同一局域网内快速发送/接收文件
- **硬件信息**：获取并展示机器硬件/系统基础信息

## 📦 安装
1. 从 `Releases` 下载最新版本安装包（`dmg`/`zip`）
2. 将 `HiveBar.app` 拖入“应用程序”
3. 首次打开若被拦截：前往「系统设置 → 隐私与安全性」选择“仍要打开”

> 如果你有签名/公证（`notarization`），也可以在此补充说明；如果没有，也可以保留这段提示。

## 🔐 权限与安全说明
- 修改 `hosts` 属于系统敏感操作：仅在**用户主动操作并确认**后执行
- 文件分享仅在**局域网**范围内进行
- 默认不上传个人数据（如有日志/统计/联网行为，请在此明确说明）
